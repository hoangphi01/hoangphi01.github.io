1. Modular arithmetic

Basic modular arithmetic
Result


                            

                                Example: −55 ≡ 7 (mod 31)
                                

                                    Giả sử a = −55, n = 31.
                                    
Chia −55 cho 31:
                                    
−55 = −2·31 + 7.
                                    
Số dư là 7 nên:
                                    
−55 ≡ 7 (mod 31).
                                    
Khi nhập:
                                    
a = -55, b = 0, n = 31
                                    
thì ô a mod n sẽ cho kết quả 7.
                                

                            


                            

                                Dùng để ôn cộng, trừ, nhân mod n, kể cả với số âm.

GCD and inverse


                        

                            

                                
                                
                            

                            

                                
                                
                            

                            
                        


                        

                            

                                
                                
                            

                            

                                
                                
                            

                            
                        


                        

                            
Result

                            


                            

                                Example: gcd(30,18) và 7⁻¹ mod 26
                                

                                    GCD:
                                    
30 = 1·18 + 12
                                    
18 = 1·12 + 6
                                    
12 = 2·6 + 0
                                    
gcd(30,18) = 6.
                                    
Inverse:
                                    
Nhập a = 7, m = 26. Tool dùng Euclid mở rộng để tìm x sao cho 7x ≡ 1 (mod 26).
                                    
Kết quả sẽ là 15 vì 7·15 = 105 ≡ 1 (mod 26).
                                

                            


                            

                                Dùng cho Affine, RSA, Diffie Hellman.

Fast exponentiation

                            


                            
Direct BigInt

                            


                            

                                Example: 3⁴⁹ mod 11
                                

                                    Viết 49 dạng nhị phân: 49 = 32 + 16 + 1 = 110001₂.
                                    
Tính dần:
                                    
3¹ mod 11 = 3
                                    
3² mod 11 = 9
                                    
3⁴ mod 11 = 9² = 81 ≡ 4
                                    
3⁸ mod 11 = 4² = 16 ≡ 5
                                    
3¹⁶ mod 11 = 5² = 25 ≡ 3
                                    
3³² mod 11 = 3²·3¹⁶ = 9·3 = 27 ≡ 5
                                    
3⁴⁹ = 3³²·3¹⁶·3¹
                                    
→ 5·3·3 = 45 ≡ 1 (mod 11).
                                    
Nhập a=3, b=49, n=11 để xem tool làm và so với phép tính trực tiếp.
                                

                            


                            

                                Dùng cho RSA, Diffie Hellman, ElGamal.
                            

                        

                    

                

            


            
            

                

                    

                        
Order modulo n


                        

                            

                                
                                
                            

                            

                                
                                
                            

                            
                        


                        

                            
Result

                            


                            

                                Example ord₁₀₀(7) và 7⁷⁹ mod 100
                                

                                    1. gcd(7, 100) = 1 nên order tồn tại.
                                    
2. Lũy thừa:
                                    
7¹ ≡ 7 (mod 100)
                                    
7² ≡ 49 (mod 100)
                                    
7³ = 49·7 = 343 ≡ 43 (mod 100)
                                    
7⁴ = 43·7 = 301 ≡ 1 (mod 100)
                                    
3. Không có số mũ nhỏ hơn 4 cho ra 1 nên ord₁₀₀(7) = 4.
                                    
4. Rút gọn số mũ:
                                    
79 = 4·19 + 3 nên 7⁷⁹ ≡ 7³ ≡ 43 (mod 100).

Result

                            


                            

                                Example: φ(33), Euler và Fermat nhỏ
                                

                                    φ(33):
                                    
33 = 3·11 là tích hai số nguyên tố nên φ(33) = (3−1)(11−1) = 2·10 = 20.
                                    
Euler:
                                    
Chọn a = 2, n = 33, ta kỳ vọng 2²⁰ mod 33 = 1 nếu gcd(2,33)=1.
                                    
Fermat:
                                    
Chọn p = 31, a = 2. Định lý Fermat nhỏ nói 2³⁰ ≡ 1 (mod 31).
                                    
Có thể nhập các giá trị này vào form để xem tool tính.
                                

                            


                            

                                Dùng để tính khóa RSA và rút gọn lũy thừa lớn.
                            

                        

                    

                

            


            
            

                

                    

                        
Primitive root modulo p


                        

                            

                                
                                
                            

                            

                                
                                
                            

                            
                        


                        

                            
Result

                            


                            

                                Example: 3 có là generator mod 31 không
                                

                                    Với p = 31, p − 1 = 30.
                                    
Phân tích 30 = 2·3·5 nên các ước nguyên tố là 2, 3, 5.
                                    
3 là generator nếu:
                                    
3^{30/2} = 3^{15} mod 31 ≠ 1
                                    
3^{30/3} = 3^{10} mod 31 ≠ 1
                                    
3^{30/5} = 3^{6} mod 31 ≠ 1.
                                    
Tool sẽ tự tính các lũy thừa này và thông báo 3 là generator hay không.
                                

                            


                            

                                Hữu ích cho Diffie Hellman, ElGamal.
                            

                        

                    

                

            


            
            

                

                    

                        
Cipher cheat sheet

                        

                            

                                Affine: C = (a·P + b) mod n. Cần modular inverse cho a⁻¹ mod n.
                            

                            

                                Vigenere: cộng mod 26 theo key. Dùng thẻ Vigenere + Alphabet converter.
                            

                            

                                AES: XOR (AddRoundKey), nhân ma trận trong GF(2⁸), nghịch đảo trong
                                GF(2⁸).
                            

                            

                                RSA: C = Mᵉ mod n, M = Cᵈ mod n. Cần Euclid mở rộng, φ(n), lũy thừa
                                nhanh.
                            

                            

                                Diffie Hellman: K = g^{ab} mod p. Cần primitive root g, fast
                                exponentiation.
                            

                            

                                Miller Rabin: dựa trên việc kiểm tra x² ≡ 1 mod n có nghiệm không tầm
                                thường.
                            


                            

                                Example: dùng toolkit để giải RSA nhỏ
                                

                                    1. Chọn p, q nhỏ và tính n = p·q, φ(n) bằng thẻ Euler phi.
                                    
2. Chọn e sao cho gcd(e, φ(n)) = 1 dùng thẻ GCD.
                                    
3. Dùng thẻ inverse để tính d = e⁻¹ mod φ(n).
                                    
4. Mã hóa và giải mã M dùng thẻ Fast modular exponentiation.
                                

                            


                            

                                Lời khuyên: ưu tiên thành thạo Euclid mở rộng và lũy thừa nhanh.

Decimal and binary


                        

                            
Decimal to binary

                            

                                
                                
                            

                            
                        


                        


                        

                            Example: 13 → 1101₂
                            

                                13 = 8 + 4 + 1 = 2³ + 2² + 2⁰.
                                
Nên dạng nhị phân là 1101.
                                
Nhập 13, tool sẽ in 1101.
                            

                        


                        


                        

                            
Binary to decimal

                            

                                
                                
                            

                            
                        


                        

                            


                            

                                Example: 1010₂ → 10
                                

                                    1010₂ = 1·2³ + 0·2² + 1·2¹ + 0·2⁰
                                    = 8 + 0 + 2 + 0 = 10.
                                    
Nhập 1010, tool sẽ trả về 10.
